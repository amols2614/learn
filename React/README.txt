##

	open command promt with path where you want to create your first react app 

	ex- c:\react\

	then run below command to create app 
	npx create-react-app my-app

	above command may take few minutes 

	then enter to the app directory by below command 
	cd my-app

	and run below command to run your first application

	npm start

	it will simply start your first react app which you just created
